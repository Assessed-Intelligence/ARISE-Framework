# ARISE Crosswalks

Mappings between ARISE Framework™ controls and 40 external frameworks, regulations, and audit criteria. One CSV per source. Each row links an ARISE control identifier to the corresponding clause, article, or control in the external standard, with the mapping rationale.

These mappings indicate conceptual alignment; they are not certifications of compliance with any external standard. Evidence requirements and supplemental control guidance per mapping are available through Assessed Intelligence engagements and commercial licensing. External standards remain the property of their respective publishers; adopters must consult the authoritative source documents.

## Schema

Every file contains `arise_control_id`, the external reference column(s) from the source standard, and `rationale` describing why the mapping holds.
