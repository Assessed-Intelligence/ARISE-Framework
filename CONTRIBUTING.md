# Contributing

The ARISE Framework™ is a controlled publication of Assessed Intelligence LLC. The framework structure, control taxonomy, and mappings are maintained by Assessed Intelligence; this repository does not accept pull requests that modify the framework itself.

## How to Contribute

- **Report a mapping error or ambiguity**: open a GitHub issue with the control ID, the external standard reference, and the correction you propose.
- **Suggest a new crosswalk**: open an issue describing the standard, the audience that needs the mapping, and any regulatory driver.
- **Share adoption feedback**: issues describing how ARISE is being applied in education, research, or internal governance help prioritize future releases.

Accepted corrections are incorporated by the maintainers and credited in the release notes.
